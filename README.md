## Getting started

This project requires Rust to be installed. On Linux or OSX, you can install rust with the following: 

```console
curl https://sh.rustup.rs -sSf | sh
```

Running it then should be as simple as:

```console
$ make
$ ./bin/assignment_one
```

### Testing

``make test``
